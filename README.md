# The Capture: A Multi-Format Archive

## Contents

This archive contains the same truth encoded in multiple formats for resilience:

| File | Format | Purpose |
|------|--------|---------|
| `the_capture.html` | HTML | Self-contained, viewable in any browser, includes all tabs |
| `chronicle.md` | Markdown | Full documented timeline, citable, versionable |
| `parable.md` | Markdown | Allegorical encoding for retelling |
| `data/chronicle.json` | JSON | Machine-readable structure for embedding |

## Quick Start

1. Open `the_capture.html` in any web browser
2. Navigate between Chronicle, Parable, Data, Maxims, and Distribution tabs
3. Use the Download button to save a copy
4. Follow distribution instructions

## Distribution Protocol

**Principle:** Redundancy defeats deletion. Distribution defeats capture.

### Locations
- Personal storage devices
- Git repositories (fork this)
- IPFS (content-addressed)
- Archive.org
- Friend networks
- Print copies

### Actions
1. Save locally
2. Email to trusted contacts
3. Upload to git repository you control
4. Submit to Archive.org
5. Print a copy
6. Tell the parable
